Implementation of the W3C RDF Dataset Canonicalization algorithm (RDF-Canon) for Dart.

## Features

- Deterministic serialization of RDF datasets
- Consistent blank node identification
- W3C RDF Dataset Canonicalization compliant
- Cryptographic hashing support

## Installation

```yaml
dependencies:
  locorda_rdf_canonicalization: ^0.1.0
```

## Use Cases

- Dataset comparison and deduplication
- Cryptographic signing of RDF data
- Version control for RDF datasets
- Consistent dataset hashing
