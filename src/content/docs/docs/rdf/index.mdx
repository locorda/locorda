# RDF Libraries for Dart

Documentation for Locorda's RDF libraries is coming soon.

## Available Packages

- **[rdf_core](https://pub.dev/packages/rdf_core)** - Parse, query, and manipulate RDF data
- **[rdf_canonicalization](https://pub.dev/packages/rdf_canonicalization)** - RDF dataset canonicalization (RDF-C14N)
- **[rdf_mapper](https://pub.dev/packages/rdf_mapper)** - Map Dart objects to RDF and back
- **[rdf_xml](https://pub.dev/packages/rdf_xml)** - RDF/XML support
- **[rdf_vocabularies](https://pub.dev/packages/rdf_vocabularies)** - Common RDF vocabularies for Dart
